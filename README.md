
# Vowel Gender Classification

This repository contains code for data preprocessing,
normalization, Leave-One-Speaker-Out classification, and
evaluation used in the ACL submission.

Usage:
1. Install requirements
2. Run `f.py` to reproduce results

The dataset is not included due to licensing/privacy.
Expected CSV format:

Columns:
- Speaker
- Gender
- Vowel
- F1
- F2
- F3

Results are shown in the paper.
